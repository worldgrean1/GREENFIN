Usage Guidelines

Best practices and implementation guidelines for maintaining brand consistency across all touchpoints.

Do's

  • Use the primary green (#3DD56D) for main CTAs and brand elements
  • Maintain proper contrast ratios for accessibility compliance
  • Use official patterns and gradients for visual consistency
  • Implement both light and dark theme variations
  • Follow the 60-30-10 color distribution rule

Don'ts

  • Don't modify the official green color values or create custom variations
  • Don't use the logo on backgrounds with insufficient contrast
  • Don't stretch, distort, or modify the official logo proportions
  • Don't mix unofficial fonts with the brand typography system
  • Don't ignore accessibility guidelines for color contrast
